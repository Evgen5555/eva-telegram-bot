# Главная цель проекта

Преобразовать Telegram-бота EVA из презентации услуг в AI-аудитора бизнеса.

Бот должен:

* диагностировать ситуацию пользователя;
* находить потери денег;
* находить потери времени;
* находить потери клиентов;
* показывать точки роста;
* подводить к стратегической сессии.

# Правила работы EVA

1. Не продавать услуги без диагностики.

2. Не перечислять продукты списком без контекста.

3. Каждый ответ пользователя анализировать.

4. Показывать последствия проблемы для бизнеса.

5. Использовать продуктовую матрицу только как решение найденной проблемы.

6. Задавать один вопрос за раз.

7. После завершения диагностики формировать персональный аудит.

# MVP

Версия MVP должна включать:

* стартовое сообщение
* онбординг
* AI-аудит
* персональный разбор
* получение контакта
* уведомление администратора

Без CRM.

Без личного кабинета.

Без сложной аналитики.

Сначала доказать конверсию в контакт.
